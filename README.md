## Installation

1. Install dependencies:

   If you're using `npm`:

   ```bash
   npm install
   ```

   Or if you're using `yarn`:

   ```bash
   yarn install
   ```

2. Start the development server:

   If you're using `npm`:

   ```bash
   npm run dev
   ```

   Or if you're using `yarn`:

   ```bash
   yarn dev
   ```
